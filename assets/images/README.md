# Profile Image

Drop your profile image here as `profile.jpg` and it will automatically appear in the sidebar.

**Specs:**
- Filename: `profile.jpg` (or `profile.png`)
- Size: 400x400px (square)
- Format: JPG or PNG
- File size: Under 200KB recommended

The image will be displayed as a perfect circle with an indigo border in the sidebar above your name.

---

To add your image:
```bash
# Simply copy your image here
cp /path/to/your/photo.jpg profile.jpg
```

Or just drag & drop `profile.jpg` into this folder.
